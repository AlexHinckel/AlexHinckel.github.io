VARIAVEIS

Existem dois casos de escopo de variáveis.

Variaveis globais

São reconhecidas, ou seja, seu escopo compreende todo o programa.

numero = 42 # Exemplo de variavel global

Variaveis locais

São reconhecidas apenas no bloco onde foram declaradas, ou seja, seu escopo esta limitado ao bloco onde foi declarada.

numero = 2
if numero > 10:
    novo = numero +10 # A variavel 'novo' esta declarada localmente dentro do bloco do if. Portando é local.
    print(novo)

Python é uma linguagem de tipagem dinâmica. Isso significa ao declararmos uma variavel,nós não colocamos o tipo de dado dela. Esse tipo é inferido ao atribuirmos o valor a mesma.

Exemplo em Java:
int numero = 42;

OPERADORES LOGICOS (AND, OR e NOT)

Utilizamos os operadores lógicos quando precisamos realizar operações sobre um ou dois valores booleanos.

Estruturas lógicas: and, (e), or (ou), not (não), is (é)

Considere que a variável A contém o valor true (verdadeiro) e a variável B contém false (falso).

Operador Descrição Exemplo
and Operador AND (E) (A and B) é false
or Operador OR (OU) (A or B) é true
not Operador NOT (NÃO) ou Negação not(A and B) é true
is Operador is (é) (A is B) é false

Operadores unários:

  • not

Operadores Binários

  • and
  • or
  • is

Regras de funcionamento:

  • Para o ‘and’, ambos os valores precisam ser True.
  • Para o ‘or’, um ou outro valor precisa ser True.
  • Para o ‘not’, o valor do booleano é invertivo.
  • Para o ‘is’, o valor é comparado com um segundo.

ENTENDENDO RANGES

Precisamos conhecer o loop for para usar o ranges e conhecer melhor o range para trabalhar com for.

Ranges são utilizados para gerar sequencias numericas, não de forma aleatoria, mas de maneira especificada.

range (valor_inicio,valor_de_parada,passo)

OBS:

  • valor de parada não inclusive
  • inicio padrão 0
  • passo padrão de 1 em 1

Formas gerais:

#Forma 1
for num in range(11):
    print(num)

#Forma 2
for num in range (5, 11):
    print(num)

#Forma 3
for num in range (0, 11, 2):
    print(num)

#Forma 4 (Inversa)
for num in range (10, 0, -1):
    print(num)

LISTAS

Listas em Python funcionam como vetores/matrizes (arrays) em outras linguagens, com a diferença de serem dinâmicos e também de podermos colocar qualquer tipo de dado.

  • Dinâmico: não possuem tamanho fixo; Ou seja, podemos criar a lista e simplesmente adicionar novos elementos.
  • Qualquer tipo de dado: as listas não possuem tipo de dado fixo; Ou seja, podemos colocar qualquer tipo de dado

As listas em python são representadas por colchetes []

PyInstaller - Gerando executaveis

Fonte: https://pypi.org/project/pyinstaller

PyInstaller agrupa um aplicativo Python e todas as suas dependências em um único pacote. O usuário pode executar o aplicativo empacotado sem instalar um interpretador Python ou quaisquer módulos.

PyInstaller lê um script Python escrito por você. Ele analisa seu código para descobrir todos os outros módulos e bibliotecas de que seu script precisa para ser executado. Em seguida, ele coleta cópias de todos esses arquivos - incluindo o interpretador Python ativo! - e os coloca com seu script em uma única pasta ou, opcionalmente, em um único arquivo executável.

PyInstaller foi testado em Windows, Mac OS X e GNU / Linux. No entanto, não é um compilador cruzado: para fazer um aplicativo do Windows, você executa o PyInstaller no Windows; para fazer um aplicativo GNU / Linux, você pode executá-lo em GNU / Linux, etc. O PyInstaller foi usado com sucesso com AIX, Solaris, FreeBSD e OpenBSD, mas não foi testado com eles como parte dos testes de integração contínua.

Instalação e geração .EXE

pip install pyinstaller
pyinstaller --onefile Arquivo.py
### Opção de execução em background
pyinstaller --onefile --noconsole Arquivo.py
